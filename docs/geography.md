# FLXguide Geographic Definition

## Overview

FLXguide covers the Finger Lakes region of New York State, defined as 13 counties (based on the official Tourism Council definition, with modifications to exclude major metro areas that have their own distinct identity).

This document serves as the **source of truth** for all data collection and content boundaries.

---

## The 13 Counties

| County | Key Towns/Villages | Key Features |
|--------|-------------------|--------------|
| Cayuga | Auburn, Union Springs, Aurora | Owasco Lake, Cayuga Lake (east shore) |
| Chemung | Elmira, Horseheads | Southern gateway, Mark Twain connection |
| Cortland | Cortland, Homer | Eastern edge |
| Livingston | Geneseo, Dansville, Livonia | Conesus Lake, Hemlock Lake, Canadice Lake, Letchworth State Park |
| Onondaga | Skaneateles, Marcellus, Spafford | Skaneateles Lake, Otisco Lake - **Excludes Syracuse metro** |
| Ontario | Canandaigua, Geneva, Naples, Victor | Canandaigua Lake, Honeoye Lake |
| Schuyler | Watkins Glen, Montour Falls, Hector | Seneca Lake (south), Watkins Glen State Park |
| Seneca | Seneca Falls, Romulus, Ovid | Seneca Lake, Cayuga Lake, Women's Rights history |
| Steuben | Corning, Hammondsport, Bath | Keuka Lake (south), Corning Museum of Glass |
| Tioga | Owego, Waverly | Southern gateway |
| Tompkins | Ithaca, Trumansburg, Dryden | Cayuga Lake (south), Cornell, gorges & waterfalls |
| Wayne | Sodus, Palmyra, Newark | Northern border, Lake Ontario, Erie Canal |
| Yates | Penn Yan, Dundee, Branchport | Keuka Lake |

## Exclusions

- **Monroe County / Rochester** - Major metro with its own identity
- **Syracuse metro area** - Major metro with its own identity (Skaneateles/Otisco area IS included)

## Usage Notes

When collecting data (wineries, restaurants, campgrounds, boat launches, etc.), use these 13 counties as the geographic filter. For searches, any establishment located within these county boundaries should be included.

### Data Collection Guidelines

1. **Primary filter**: County boundaries as defined above
2. **Secondary consideration**: Proximity to Finger Lakes attractions
3. **Edge cases**: When in doubt, include if the establishment primarily serves Finger Lakes visitors

### Search Queries

For Google Maps, Yelp, or other data sources, search by:
- Individual county names + category
- Town/village names listed above
- "Near [Lake Name]" for lake-specific searches
